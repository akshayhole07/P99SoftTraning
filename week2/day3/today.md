4. usecase of constructor and destructor in python:- 
Constructor: A constructor is a special method in Python that is called when an object is created. It is used to initialize the attributes of the object and set up any necessary resources. The constructor is defined using the __init__ method.
- Destructor: A destructor is a special method in Python that is called when an object is about to be destroyed. It is used to perform cleanup operations before the object is removed from memory. The destructor is defined using the __del__ method.

5. snowflake schema:- A snowflake schema is a type of database schema that organizes data into a hierarchical structure. It is called a snowflake schema because the diagram of the schema resembles a snowflake. In a snowflake schema, the fact table is connected to multiple dimension tables, which are further connected to other dimension tables. This allows for more efficient storage and querying of data, as it reduces redundancy and improves data integrity.
